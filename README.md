# Better-Nether-Bricks-datapack

datapack by KillerMarce!

recipes list:
- nether brick: 4 netherrack in a 4x4 now will result in 2 nether bricks. alternatively, smelting in a furnace like vanilla will yield 1 netherrack for 1 brick, so pick whichever is better for you!
- nether bricks: 4 nether brick's like vanilla will now output 4 nether bricks instead of 1, or smelting red nether bricks in a regular furnace will turn it back into a nether bricks block.
- red nether bricks: 1 nether bricks block and 1 nether wart now will result in 1 red nether bricks.
- nether wart block: 4 nether warts in a 4x4 now results in 1 nether wart block, making it easier to craft with a nether wart farm.
- nether wart: 1 nether wart block now yields 4 nether warts, making it now farmable in crimsom tree farms, but balanced so that nether wart farms don't become obsolete.
